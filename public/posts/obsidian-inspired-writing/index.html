Obsidian-Inspired Writing

April 30, 2025
tools writing obsidian

After using Obsidian for my personal notes for over a year, I decided to bring some of its best features to my blog. In my first post, I mentioned that I’d be using an Obsidian-inspired setup, and here I want to elaborate on what that means.

What I Love About Obsidian

Obsidian has several features that make it exceptional for note-taking and writing:

  1. Markdown-based: Everything is written in simple Markdown, making it portable and future-proof.
  2. Backlinks: The ability to see which notes link to the current note.
  3. Graph View: A visual representation of connections between notes.
  4. Easy Linking: Quick and intuitive way to link between notes.

How I’ve Implemented These Features

I’ve brought these key features to my blog:

Markdown

All posts are written in Markdown with YAML front matter, similar to how I write in Obsidian. This makes it easy to:

  • Add formatting
  • Include images
  • Write code blocks
  • Structure content with headers

Links and Connections

I can easily create connections between posts using standard Markdown links. For example, I can link to my post about backlinks and graph visualization.

When you’re reading a post, you can explore related content through the graph visualization, which shows how posts are connected.

Graph Visualization

The graph view shows connections between all posts. Nodes represent posts, and edges represent links between them. This gives a visual representation of how ideas connect across different posts.

Writing Workflow

My writing workflow is simple:

  1. Create a new Markdown file with front matter
  2. Write content using Markdown
  3. Include links to connect to other posts
  4. Add the post to the index.json file
  5. Upload everything to the server

Benefits

This approach has several benefits:

  • Lightweight: The entire system is just HTML, CSS, and JavaScript. No heavy frameworks or databases.
  • Portable: All content is in Markdown files that can be easily moved or migrated.
  • Connected: Graph visualization helps connect ideas across posts.
  • Mobile-friendly: The responsive design works well on all devices.

If you’re interested in seeing how the graph visualization works in more detail, check out my post on backlinks and graph visualization.
